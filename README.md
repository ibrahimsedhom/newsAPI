## Run Locally

Clone the project

```bash
  git clone https://github.com/ibrahimsedhom/newsAPI.git
```

Go to the project directory

```bash
  cd newsAPI
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```
